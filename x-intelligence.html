X 是加分來源，不是必要條件

官方 feeds、GitHub Releases 與 arXiv 已可每天更新；設定 X Token 後再增加社群事件訊號。

到 X Developer Portal 建立 App，取得 Bearer Token。
GitHub Repo → Settings → Secrets and variables → Actions，新增 X_BEARER_TOKEN。
到 Actions 手動執行 Daily resilient AI intelligence；之後每天 08:37（台北時間）自動更新。

使用界線

新消息只決定「今天研究什麼」，不直接決定投資或正式環境升級。

  • 官方產品發布仍要用自己的任務測品質、耗時與成本；示範不等於你的工作流也會改善。
  • GitHub Releases 先在隔離分支驗證相容性；arXiv 預印本要等程式碼或獨立重現。
  • X 只作事件雷達；Crypto 與美股仍以市場雷達、監管文件、公司公告及正式行情為準。
