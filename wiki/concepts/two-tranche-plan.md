# 兩等份本金計畫

## 第 1 等份「投降接刀」(50%) — 任一成立即投
1. BTC ≤ $54k（或進 [[target-price]] 底部區 $40–52k）
2. BTC MVRV-Z ≤ 0
3. [[five-dimension-model]] 綜合溫度 ≤ −6
4. [[mstr]] mNAV ≤ 0.6x

## 第 2 等份「便宜＋確認」(50%) — 全部成立才投
1. BTC/ETH 現貨 ETF 連 ≥10 交易日淨流入
2. BTC 站回 200 日均線且做出更高低點
3. [[mstr]] 無被迫賣幣且 mNAV ≥ 1.0x
4. 綜合溫度 +2~+5

## 止盈
