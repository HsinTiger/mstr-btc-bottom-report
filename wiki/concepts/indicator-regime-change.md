# indicator-regime-change（鏈上/週期指標的體制轉變）

**一句話**：2025/10 頂部（$126k）時 Pi Cycle Top、MVRV、NUPL、Puell、Reserve Risk、RHODL **集體**未進危險區——不是單一指標壞掉，是這批指標校準的市場行為（散戶 FOMO 拋物線）本輪沒有發生。

## 失效機轉

| 指標 | 量測對象 | 為何失效 |
| --- | --- | --- |
| Pi Cycle Top（111DMA × 2×350DMA） | 上漲**角度**（加速度） | 機構按季配置 → 穩定爬升無拋物線；$40k→$126k 走 20 個月，兩線靠近又分開、從未交叉 |
| MVRV / MVRV-Z | 鏈上實現成本 vs 市值 | ETF/Strategy 的幣躺在 Coinbase Prime 託管錢包從未鏈上移動 → realized cap 系統性失真；頂部 MVRV 僅 3.8–4.2（不到歷史警戒一半） |
| Pi Cycle Bottom（150EMA × 471SMA×0.745） | 急殺式投降的下跌速度 | 參數是對 2015/2018 兩個底 curve-fit 出來的（n≈2）；若機構市場跌得慢而淺，EMA 永遠追不上交叉條件 |

## 對本 repo 框架的影響

1. **第 1 等份②（MVRV-Z ≤ 0）**：舊 regime 門檻，真實底部可能仍 > 0。建議雙軌：`MVRV-Z ≤ 0 或 6 個月滾動 Z-score 進入最低 5% 分位`。
2. **五維③週期遞減（12–14 月見底窗口）**：源自四年週期，前提已弱化。建議權重下調或放寬至 12–18 月。
3. **Pi Cycle Bottom 使用方式**：降級為加分項。觸發 = 加分；**不觸發 ≠ 底沒到**（本輪最重要的一句話）。
4. **引用外部目標價時**：凡以「四年週期十月重啟」為前提者（如 Benchmark $570 @ BTC 95k），一律標註「週期前提本輪已失效一次」。

## 仍然可信的底部證據（新 regime 適用）

- LTH 積累行為（直接行為，不依賴週期參數）
- ETF 流出枯竭度（6 月 −$4B 級流出歷史上偏底部反指標）
- 200WMA（跨 regime 存活最久的均線）
- 宏觀流動性（機構市場的真正主變數）

## 工程視角備註

用 n=2 訓練、對舊分布擬合的模型，遇到分布偏移（distribution shift）時的正確處置不是丟棄也不是照用，是：降權、加入新分布下的獨立特徵、並對「模型沉默」與「模型觸發」給不對稱的置信度——沉默的資訊量遠低於觸發。
